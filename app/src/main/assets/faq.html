Καλωσορίσατε στην Εφαρμογή Διαχείρισης ΕΤΥΠ!

Αυτή η εφαρμογή έχει σχεδιαστεί για να σας βοηθήσει να διαχειριστείτε αποδοτικά το ντουλάπι και τα είδη ΕΤΥΠ σας, καθιστώντας την εμπειρία των δρομολογίων σας πιο απλή και ευχάριστη. Παρακάτω παρατίθενται μερικά βασικά χαρακτηριστικά και οδηγίες για το πώς να χρησιμοποιείτε την εφαρμογή αποτελεσματικά.

Ξεκινώντας

1. Δημιουργία Ντουλαπιού:
- Με την εκκίνηση της εφαρμογής, θα σας ζητηθεί να δημιουργήσετε το ντουλάπι σας. Πατήστε το κουμπί 'Προσθήκη Αντικειμένου' για να ξεκινήσετε να προσθέτετε αντικείμενα που έχετε αυτή τη στιγμή στο απόθεμα σας.
- Εισάγετε το όνομα του αντικειμένου και καθορίστε την ποσότητα που έχετε. Μόλις προστεθούν, τα αντικείμενα σας θα εμφανιστούν στη λίστα του ντουλαπιού.

2. Διαχείριση Λίστας ΕΤΥΠ:
- Μεταβείτε στην ενότητα Λίστας ΕΤΥΠ για να προσθέσετε αντικείμενα που χρειάζεται να αγοράσετε. Ακριβώς όπως στο ντουλάπι, μπορείτε να προσθέσετε αντικείμενα εισάγοντας τα ονόματα και τις ποσότητές τους.
- Η λίστα ΕΤΥΠ θα σας βοηθήσει να παρακολουθείτε τι χρειάζεται να αγοράσετε στο επόμενο δρομολόγιο σας.
- Έχουμε υλοποιήσει τη δυνατότητα να μεταφέρετε μόνο τα επιλεγμένα αντικείμενα από τη λίστα σας στο ντουλάπι. Απλά επιλέξτε τα αντικείμενα που θέλετε να μεταφέρετε και πατήστε το κουμπί 'Μετακίνηση Αντικειμένων'. Αν δεν επιλέξετε κανένα αντικείμενο, θα μεταφερθούν όλα τα αντικείμενα της λίστας.

3. Διαχείριση Προκαθορισμένης Ποσότητας:
- Μεταβείτε στην ενότητα Απαιτούμενο Απόθεμα για να προσθέσετε αντικείμενα που θέλετε να θέσετε μια προκαθορισμένη τιμή. Η προκαθορισμένη τιμή, είναι αυτή που θα θέλαμε σε ιδανικές συνθήκες να έχουμε στο ντουλάπι.
- Ετσι, όταν πατήσετε την επιλογή Μεταφορά στη λίστα ΕΤΥΠ, θα καταγραφεί στη λίστα ΕΤΥΠ η υπολειπόμενη ποσότητα, την οποία την οποία αν προμηθευτούμε, θα έχουμε το ιδανικό.

4. Ενημέρωση Αντικειμένων:
- Μπορείτε εύκολα να ενημερώσετε τα αντικείμενα του ντουλαπιού σας. Πατήστε σε οποιοδήποτε αντικείμενο για να ανοίξετε το διάλογο επεξεργασίας, όπου μπορείτε να αλλάξετε το όνομα ή την ποσότητα του αντικειμένου.
- Για τα αντικείμενα ΕΤΥΠ, η διαδικασία είναι η ίδια: πατήστε για να επεξεργαστείτε.

5. Ρύθμιση Ποσοτήτων:
- Μπορείτε να αυξήσετε ή να μειώσετε τις ποσότητες των αντικειμένων του ντουλαπιού σας χρησιμοποιώντας τα κουμπιά ' και '-' δίπλα σε κάθε αντικείμενο.
- Αν εξαντληθείτε από ένα αντικείμενο, απλά διαγράψτε το από τη λίστα του ντουλαπιού.

6. ΝΕΟ: Αναζήτηση
- Μπορείτε πλεόν να κάνετε αναζήτηση σε κάθε μία από τις αναφερόμενες λίστες!

7. Εισαγωγή και Εξαγωγή Βάσης Δεδομένων:
- Μπορείτε να εξάγετε τη βάση δεδομένων σας για να δημιουργήσετε αντίγραφα ασφαλείας. Επιλέξτε την επιλογή 'Εξαγωγή Βάσης Δεδομένων' από το μενού ρυθμίσεων για να ξεκινήσετε τη διαδικασία.
- Για να εισάγετε μια βάση δεδομένων, επιλέξτε 'Εισαγωγή Βάσης Δεδομένων' από το μενού και επιλέξτε το αρχείο που θέλετε να εισάγετε.

8. Ιστορικό Συναλλαγών:
- Η εφαρμογή περιλαμβάνει μια δυνατότητα ιστορικού συναλλαγών, επιτρέποντάς σας να παρακολουθείτε τα αντικείμενα που έχετε προσθέσει ή αφαιρέσει από το ντουλάπι σας.
- Μπορείτε να δείτε το ιστορικό συναλλαγών μεταβαίνοντας στην ενότητα 'Ιστορικό Συναλλαγών'.

Πρόσθετα Χαρακτηριστικά

- Φιλικό Περιβάλλον Χρήστη: Η εφαρμογή έχει σχεδιαστεί ώστε να είναι διαισθητική και φιλική προς τον χρήστη, εξασφαλίζοντας ότι μπορείτε να περιηγηθείτε μέσω του ντουλαπιού και των λιστών ΕΤΥΠ σας με ευκολία.

Συμβουλές για Καλύτερη Χρήση

- Διατηρήστε τις Λίστες σας Ενημερωμένες: Ενημερώνετε τακτικά τις λίστες ντουλαπιού και ΕΤΥΠ σας ώστε να αντικατοπτρίζουν οποιεσδήποτε αλλαγές στο απόθεμά σας. Αυτό θα σας βοηθήσει να αποφύγετε διπλές αγορές.
- Εξερευνήστε την Εφαρμογή: Πάρτε λίγο χρόνο για να εξερευνήσετε όλα τα χαρακτηριστικά που προσφέρει η εφαρμογή, συμπεριλαμβανομένου του ιστορικού συναλλαγών και των ρυθμίσεων χρήστη.
- Ανατροφοδότηση: Εκτιμούμε την ανατροφοδότηση σας! Εάν αντιμετωπίσετε οποιοδήποτε πρόβλημα ή έχετε προτάσεις για βελτιώσεις, παρακαλούμε επικοινωνήστε μαζί μας μέσω της εφαρμογής.
